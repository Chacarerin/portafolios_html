Primera versión del portafolio web personal. Implementado íntegramente con tecnologías web nativas sin dependencias de frameworks o librerías externas.

## 🧠 Contexto Pedagógico y Teórico
Construir un portafolio desde cero sin frameworks es la prueba más directa de dominio de los fundamentos web. Esta versión prioriza la **claridad arquitectónica**: un archivo HTML para la estructura semántica del documento, un archivo CSS para la capa de presentación y estilos responsivos, y un archivo JavaScript para el comportamiento interactivo. Esta separación de responsabilidades (SoC) es el principio de diseño más fundamental del desarrollo front-end moderno.

## ⚙️ Tecnologías y Frameworks Aplicados
* **HTML5 Semántico**: Estructura del documento basada en etiquetas con semántica clara para motores de búsqueda y lectores de pantalla.
* **CSS3 con assets organizados**: Los estilos se gestionan en `assets/css/style.css`, separando la presentación del contenido. Se aplican técnicas de diseño responsivo para garantizar compatibilidad con múltiples tamaños de pantalla.
* **JavaScript Vanilla (`assets/js/script.js`)**: Interactividad del cliente implementada con JS puro, sin jQuery ni librerías, demostrando dominio de la API del DOM.

## 🛠️ Desglose Técnico (El "Cómo")
* **`index.html`**: Página principal del portafolio con secciones de presentación, habilidades y proyectos.
* **`form.html`**: Formulario de contacto con validación de campos en el lado del cliente.
* **`assets/`**: Directorio de recursos estáticos con imágenes, hojas de estilo y scripts organizados por tipo.
