Este repositorio concentra distintas versiones del portafolio web personal, desarrolladas con tecnologías web estándar. Documenta la evolución del diseño desde una primera versión construida manualmente hasta una versión generada con asistencia de Inteligencia Artificial.

## 🧠 Contexto Pedagógico y Teórico
El portafolio web es una herramienta fundamental para comunicar competencias técnicas a reclutadores y pares. Este repositorio evidencia el dominio de los fundamentos del desarrollo web front-end sin dependencias de frameworks, lo que demuestra un conocimiento profundo de las tecnologías base del ecosistema web.

## 📁 Índice de Versiones
* **[`portfolio_html_v1/`](./portfolio_html_v1/)**: Primera versión del portafolio, construida con HTML semántico, CSS personalizado y JavaScript puro, siguiendo principios de diseño responsivo.
* **[`portfolio_html_AI/`](./portfolio_html_AI/)**: Versión generada con asistencia de Inteligencia Artificial, explorando flujos de trabajo de desarrollo aumentado por IA.

## ⚙️ Tecnologías y Metodologías Aplicadas
* **HTML5 Semántico**: Uso de etiquetas estructurales (`<section>`, `<article>`, `<header>`, `<nav>`) para mejorar la accesibilidad (a11y) y el posicionamiento en buscadores (SEO), en contraste con el enfoque antiguo de `<div>` genéricos.
* **CSS3 Vanilla**: La decisión de usar CSS puro frente a frameworks como TailwindCSS o Bootstrap permite demostrar dominio real del modelo de caja, flujo de documento, flexbox y media queries.
